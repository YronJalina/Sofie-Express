# Sofie Express Website

This is the **official website for Sofie Express**, a personal flight delivery service between **Dubai ↔ Manila**.  

The site is fully **responsive, minimalist, and professional**, with features including:

- **Hero Section:** Manila ↔ Dubai collage background with soft parallax and subtle CGI depth  
- **Flight Schedule Board:** Dynamic display highlighting the next flight and available KG  
- **Rates Section:** Shows AED per KG (35 AED/kg)  
- **Locations Section:** Dubai & Manila (Rizal near Metro Manila)  
- **Booking Form:** Fully functional via Formspree  
- **Custom Cursor:** Trail particles that respond to hover events  
- **Smooth Scroll Navigation**  
- **Responsive Design:** Works on mobile, tablet, and desktop
